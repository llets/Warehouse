    Автоматизация заполнения БД из файлов JSON
1. Скачиваем Postman.
2. Скачиваем мою коллекцию warehouse fill fields.
3. Кликаем на ней 2 раза и переходим во вкладку Runs.
4. Кликаем Run collection.
5. Выбираем один из запросов, выбираем соответствующий файл (по названию).
6. Устанавливаем количество итераций = количеству объектов в файле.
7. Нажимаем Run warehouse fill fields
8. Убеждаемся, что всё прошло успешно.
9. Повторяяем шаги 5-8 для всех таблиц.

* перед этим обязательно включить сервер:
cd server
npm run dev
